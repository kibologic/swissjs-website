# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | yes       |

## Reporting a Vulnerability

To report a security vulnerability in the SwissJS framework or this website:

1. **Do not open a public GitHub issue.**
2. Email security@kibologic.com with a full description.
3. Include steps to reproduce and the potential impact.
4. We aim to respond within 48 hours.

## Scope

This policy covers the swissjs-website source code. For vulnerabilities in the SwissJS runtime or compiler packages (`kibologic/swiss-lib`), please report to security@kibologic.com.
